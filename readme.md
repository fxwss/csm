


# Steps to run

- install and add to path system dependencies
  - ffmpeg

- Install dependencies 
```bash
  npm ci
```
- Run migrations
```bash
  noce ace migration:run
```
- Run seeders
```bash
  node ace db:seed
```

- Start the server
```bash
  npm run dev
```
